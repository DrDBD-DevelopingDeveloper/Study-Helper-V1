# Study Checklist App (deployable)

This is a small React app (Create React App) containing the Integrated Study & Cut checklist you asked for.
It includes:
- Daywise checklist (Nov 24 → Dec 17)
- Strict timetable generator (Nov 25 → Dec 16)
- Exam completion toggles that reallocate study time
- Downloads of PDFs and mess menu (placed in public/)

### To run locally

1. Install Node.js 18+ and npm.
2. In project folder run:
   ```bash
   npm install
   npm start
   ```
3. The app will be available at https://study-helper-v1-dbd.vercel.app/

### To deploy quickly
- Recommended: Vercel or Netlify. See instructions in the companion note (below).

Files included in public/: the PDFs and mess image you uploaded. If any file is missing, place it in public/ before building.
